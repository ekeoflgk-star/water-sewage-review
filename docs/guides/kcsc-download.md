## 2. KCSC 다운로드 절차

### Step 1: 사이트 접속
```
https://www.kcsc.re.kr/
```

### Step 2: 건설기준 검색
1. 상단 메뉴 **"건설기준"** 클릭
2. **"표준시방서(KCS)"** 선택
3. 왼쪽 분류에서:
   - **[57] 상수도** → 상수도 관련 KCS 목록
   - **[61] 하수도** → 하수도 관련 KCS 목록
   - **[14] 콘크리트** → 콘크리트 관련 KCS
   - **[11] 토공** → 토공사 관련 KCS

### Step 3: 개별 기준 다운로드
1. 해당 코드 클릭 (예: KCS 57 10 05)
2. 상세 페이지에서 **"다운로드"** 또는 **"한글파일(HWP)"** 클릭
3. HWP 파일 저장

> **참고**: KCSC는 HWP 형식만 제공합니다. PDF 직접 다운로드는 불가합니다.

### Step 4: 파일 저장 위치
```
D:\우종현\100.AI실무 아이템\01-2.설계 품질 검토\water-sewage-review\data\kcs-hwp\

파일명 규칙:
  KCS_57_10_05_상수도공사_일반사항.hwp
  KCS_57_30_00_관로_부설공사.hwp
  KCS_61_10_00_하수도공사_일반사항.hwp
  ...
```

## 3. HWP → PDF 변환

### 방법 A: 한/글에서 직접 변환 (권장 ★★★)

```
1. 한/글 프로그램에서 HWP 파일 열기
2. [파일] → [다른 이름으로 저장] → 파일 형식: PDF
3. 저장 위치: data/kcs-pdf/
4. 파일명: KCS_57_10_05_상수도공사_일반사항.pdf
```

**품질**: ★★★★★ (서식·표·그림 완벽 보존)
**소요시간**: 종당 ~2분, 14종 = ~30분

### 방법 B: 한/글 배치 변환 (대량)

한/글에서 여러 파일을 한꺼번에 변환:
```
1. [파일] → [일괄 변환] (한/글 2020 이상)
2. 원본 폴더: data/kcs-hwp/
3. 변환 형식: PDF
4. 출력 폴더: data/kcs-pdf/
```

### ⚠️ 주의: LibreOffice 변환은 비추천

LibreOffice `soffice --convert-to pdf`는 한글 서식이 깨질 수 있습니다.
특히 시방서의 복잡한 표와 수식이 정확히 변환되지 않을 위험이 있습니다.

## 4. 변환 후 확인

### 파일 구조 확인
```
data/
├── kcs-hwp/              ← KCSC 원본 HWP (백업 보관)
│   ├── KCS_57_10_05_상수도공사_일반사항.hwp
│   ├── KCS_57_30_00_관로_부설공사.hwp
│   └── ...
└── kcs-pdf/              ← PDF 변환본 (뷰어 입력용)
    ├── KCS_57_10_05_상수도공사_일반사항.pdf
    ├── KCS_57_30_00_관로_부설공사.pdf
    └── ...
```

### PDF 품질 확인
- [ ] 표(테이블)가 정확히 보이는지
- [ ] 그림/도면이 포함되어 있는지
- [ ] 글자 깨짐 없는지
- [ ] 페이지 번호가 정확한지

## 5. 다음 단계

PDF 변환이 완료되면, 다음 스크립트로 WebP 이미지 변환을 진행합니다:

```bash
node scripts/convert-doc-images.js data/kcs-pdf/
```

이후 Supabase Storage 업로드:
```bash
node scripts/upload-doc-images.js
```
