# 상하수도 설계 성과품 AI 검토 플랫폼 — 전체 브리핑

> 이 문서는 프로젝트의 모든 기획·설계·조사 결과를 통합 정리한 것이다.
> 다른 Claude 세션에서 이 문서 하나만 읽으면 프로젝트를 이어갈 수 있다.

---

## 1. 프로젝트 개요

### 1.1 목적
- 상하수도 설계 성과품(PDF·DOCX·XLSX)을 업로드하면 법령·KDS 설계기준과 자동 대조해 적합/부적합 판정
- 회사 부서 내 사용 (회원제·구독제 없음), 담당자 개인 실적용
- 사용자는 URL 접속만 하면 됨 (Claude.ai 계정 불필요)

### 1.2 현재 상태
- 기획·설계 완료, 검토 기준 조사 완료
- 아직 코드 작성 시작 전 — Phase 1 첫 번째 작업부터 시작 예정

## 5. 개발 단계 (16주)

### Phase 1 (1~3주): 환경세팅 + UI 골격 + 파일 파싱 + Gemini 채팅 연동
- 완료기준: PDF 업로드 후 문서 내용 기반 Q&A 작동
- "검토기준문서" 그룹을 UI에 미리 반영

### Phase 2 (4~7주): KDS → pgvector 임베딩 + RAG 검토 엔진 + 인허가 검토
- 완료기준: 수리계산서 업로드 후 KDS 조항 인용하며 자동 판정
- Supabase 테이블: `knowledge_base`(사전 DB) + `project_documents`(사용자 업로드 임시)
- 인허가 자동 판단 로직 구현

### Phase 3 (8~11주): 법제처 Open API 연동 + 파도타기 탐색 UI
- 법제처 API: open.law.go.kr (무료, 수도법·하수도법 조문 실시간 조회)
- 지자체 조례 실시간 조회 (하수도 사용 조례, 도로 점용 조례 등)
- 파도타기: 법률→시행령→설계기준→지침 계층 클릭 탐색

### Phase 4 (12~16주): 검토보고서 PDF 출력 + NextAuth 인증 + 성능 최적화

- codil.or.kr (CODIL): 환경부 고시·지침 PDF, 무료
- 환경부 직접 발간: 매뉴얼·실무요령, 검색 필요, 무료

### Layer 2 — 사용자 업로드 문서 (세션/프로젝트 기반, 임시 저장)
사용자가 직접 올리는 문서. 해당 프로젝트 내에서만 유효.

**대상:**
- 해당 지자체 하수도정비 기본계획 (20년 단위 계획서, 200~500p)
- 서울시 소구역 가이드라인 같은 비공개 지자체 지침
- 해당 사업 과업지시서
- 지자체 고유 설계 조건이 포함된 문서

**기술 구현:**
- 업로드 → 파싱 → 청크 분할 → Supabase `project_documents` 테이블에 임시 벡터 저장
- RAG 검색 시 Layer 1 + Layer 2 동시 검색
- 검색 결과에 출처 태그 명시: [KDS 61 40 10 §3.2] vs [업로드: 김천시 기본계획 p.45]
- 프로젝트 종료 시 임시 데이터 삭제

### Layer 3 — 법제처 Open API (Phase 3, 실시간 조회)
- 법령·조례 실시간 조회 (임베딩 불필요)
- 해당 지자체 하수도 사용 조례, 수도 급수 조례, 도로 점용 조례 자동 조회
- 우측 법령 참조 패널에 표시

## 7. 설계 검토 항목 (155개 항목, 7개 분야)

엑셀 파일 "상하수도_설계검토_체크리스트.xlsx"로 별도 정리됨. 주요 구조:

### 7.1 하수도 관로시설 (45항목)
- 계획 일반: 오수량/우수량 산정, 강우강도, 유출계수, 재현기간, 합류식 차집량
- 관거 설계: 최소관경(오수200mm↑, 우수250mm↑), Manning공식, 조도계수, 관종, 단면형상
- 유속: 오수 0.6~3.0m/s, 우수 0.8~3.0m/s, 자정유속(소류력)
- 경사: 관경별 최소경사, 역경사 불가, 급경사 감세대책
- 매설: 최소토피(차도1.0m↑, 보도0.6m↑), 기초, 되메우기, 이격거리
- 맨홀: 설치간격(600mm미만→75m, 600mm이상→150m), 내경, 접합방식, 단차
- 우수토실·차집: 합류식 차집배수
- 역사이펀, 토구, 내진, 기존관로 보수(CCTV등급, 통수능, 공법 적정성)
- 수리계산: Manning공식, 충만도(오수≤0.8, 우수≤1.0), 여유율
- 법적 사항: 기본계획 부합, 도로점용, 하천횡단

### 7.2 하수도 펌프장시설 (20항목)
- 양수량, 양정, 펌프대수, 집수정, 스크린
- NPSH, 수격작용, 밸브, 비상발전기
- 악취/소음/침수 방지, 내진

### 7.3 하수도 수처리시설 (20항목)
- 침사지, 최초침전지, 포기조(HRT/SRT/F:M비/MLSS)
- 최종침전지, 고도처리(질소/인), 여과, 소독
- 수리종단, 수밀콘크리트

### 7.4 하수도 슬러지처리 (15항목)
- 발생량, 성상, 농축, 소화(HRT/OLR/가온), 소화가스
- 탈수(함수율), 건조/소각, 반류수

### 7.5 상수도 취정수시설 (15항목)
- 취수량, 원수수질, 도수관, 혼화/응집(G값), 침전, 급속여과
- 소독(CT값), GAC, 막여과, 배출수

### 7.6 상수도 송배급수시설 (20항목)
- 배수지(12시간분↑), 관망해석(EPANET), 최소동수압 150kPa
- 소화전, 급수방식, 잔류염소 0.1mg/L↑, 내진, 제수밸브

### 7.7 공통 구조·전기·토목 (20항목)
- 하중조합, 콘크리트(fck≥24MPa), 기초, 내진, 부력(안전율≥1.2)
- 토공, 도로복구, 흙막이, 전기, TMS, 수량산출, 도면 정합성

- 근거 법조문을 사전 DB에서 인용하여 함께 표시

## 9. 지자체별 문서 현황

### 핵심 발견: 설계 기준 자체는 전국 동일 (KDS)
- 지자체가 독자적으로 유속·관경 등 설계 수치를 변경할 권한 없음
- 지자체 고유 문서는 조례(요금·행정), 하수도정비 기본계획(계획량·구역), 도로굴착 규칙 정도

### 서울시 (고유 지침 가장 많음)
- 서울특별시 하수도 사용 조례 (서울특별시조례 제9682호) → law.go.kr
- 서울특별시 도로굴착·복구업무 처리규칙 (서울특별시규칙 제4283호) → law.go.kr
- 서울시 전문시방서 SMCS (하수도: SMCS 61 XX XX) → 과업지시서 첨부
- 2040 하수도정비 기본계획 → opengov.seoul.go.kr
- 소구역 하수관로 정비사업 가이드라인 (5차) → **비공개**, 설계용역 과업지시서에 첨부 배포
- → 사용자가 Layer 2로 직접 업로드하여 사용

### 경상북도 김천시·안동시·청송군·영양군
- 각 지자체 하수도 사용 조례 → law.go.kr / elis.go.kr
- 각 지자체 수도 급수 조례 → law.go.kr / elis.go.kr
- 각 지자체 하수도정비 기본계획 → 지자체 환경과 보관, 환경부 승인본
- 독자 설계 지침은 **없음** (환경부 기준 + KDS 그대로 적용)
- → 하수도정비 기본계획만 사용자가 Layer 2로 업로드하면 충분

### 조례의 설계 영향
- 하수도 사용 조례: 주로 요금·행정 → 설계 수치 영향 거의 없음
- 도로 굴착·복구 규칙: 복구 기준(포장 두께, 다짐도) → 설계에 직접 관련
- → Phase 3에서 법제처 API로 실시간 조회하여 참조 표시
